REDESIGN PROPOSAL · 管理画面(/admin)
「魅せるゲームUI」から「静かな運用コンソール」へ
ゲーム本編（ダッシュボード・レース・LP）はネオン/サイバーで正解です。しかし /admin は運営が毎日“作業”する社内ツール。 前回はその管理画面にも消費者向けの1cネオン言語を当ててしまい、装飾がデータの視認性を奪っていました。 今回は可読性・一覧性・走査性を最優先に、ダーク基調はそのままで“落ち着いた業務ダッシュボード（Linear / Stripe 系）”へ作り替える提案です。 機能・データ・API・型は一切変更しません。
■ 現行の問題（実測）
  • 全行・全カードに装飾グラデ+ネオングロー → データより装飾が目立つ
  • ラベル8.5〜11px+広い字間、見出し用Orbitronをデータに使用 → 読みにくい
  • 行が“カード”で 列が揃わない（flex-wrap）→ 金額/日付を縦に走査できない
  • 絵文字メニュー（💰👤🎒…）→ 業務ツールに見えない
  • 低コントラスト（faint多用・薄いalpha）
■ 提案の変更点
  • 装飾グラデ/グロー廃止 → フラット面+1本の境界線。色は状態表現だけ
  • データは等幅数字+日本語サンス、字間を詰め可読サイズへ（Orbitronは見出しのみ）
  • 一覧は列見出し付きの本物のテーブル。数値は右揃え・等幅で縦に揃う
  • 絵文字を廃し、単色グリフ+明快なラベル
  • 高コントラスト化・行の余白を詰めて1画面の情報量UP
↓ 以下は実データ（プレビュー用フィクスチャ）で描いた新方向の代表画面。承認いただければ全10画面へCSS主体で展開します。

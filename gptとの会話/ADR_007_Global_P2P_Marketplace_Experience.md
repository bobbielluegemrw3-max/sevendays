# ADR-007: Global P2P Marketplace Live Experience

Status: APPROVED FOR CLAUDE CODE

## Purpose

The Daily Derby is not only a race. The Global P2P Marketplace is the
emotional climax of the daily event.

Users are most excited to discover:

-   Was my horse sold?
-   Which horse did I receive?
-   What is happening across the world?

Therefore the Marketplace sequence should receive more presentation time
than the race itself.

# Marketplace Live Sequence

## Step 1

``` text
🌍
GLOBAL MARKETPLACE OPENING...
```

Fade into a dark world map.

------------------------------------------------------------------------

## Step 2

Animated matching counter.

Example:

``` text
Matching...

8,421 / 28,122
```

Progress bar increases continuously.

------------------------------------------------------------------------

## Step 3

Light beams connect regions.

Examples:

🇺🇸 → 🇯🇵

🇧🇷 → 🇫🇷

🇩🇪 → 🇰🇷

The animation represents horses changing owners around the world.

# Global Live Ticker

Runs continuously.

Example:

``` text
🇯🇵 Royal Thunder SOLD 177.16

🇺🇸 Black Storm BOUGHT 161.05

🇧🇷 Day7 Cleared

🇩🇪 Buyback Paid

🇫🇷 Revenge Buff Generated
```

Use display names only.

Never expose wallet addresses.

------------------------------------------------------------------------

# Design Philosophy

Race ≈ 10 seconds

Marketplace ≈ 30 seconds

The Marketplace is the emotional centerpiece.

------------------------------------------------------------------------

# Sound

Minimal.

Race: soft ambience.

Marketplace: slightly faster rhythm.

Assignment: single confirmation sound.

No casino effects.

------------------------------------------------------------------------

# Constitution Rule

The Global Marketplace sequence SHALL be treated as the primary
user-facing experience of the Daily Derby.

The Marketplace animation SHALL visualize global liquidity rather than
individual transactions.

The interface SHALL emphasize:

-   Global participation
-   Successful matching
-   Horse transitions
-   Assignment completion

without exposing private financial information.
